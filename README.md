## Diseño de la base de datos

Datos que queremos guardar en la base de datos:

Clientes:



* identificador UUID
* dni
* nombre
* apellidos
* telefono
* direccion
* localidad
* fecha nacimiento

Perfil del cliente:



* credenciales de acceso a la web de banco
* situacion civil
* situacion laboral
* morosidad
* procesoJudicial
* estadoCivil
* conyuge
* enGananciales

Cuenta Bancaria:



* IBAN
* tipoDeCuenta
* saldoActual
* dineroUltimaNomina
* mediaNominas

Movimiento Cliente:



* numeroOperacion
* cantidad
* conceptoOperacion
* emisor
* destinatario

Prestamo preconcedido:



* id
* fechaPrestamo
* cantidadPrestada
* periodoPrestamo
* tipoDeInteres
* plazoDeConfirmacion

Prestamo concedido:



* fechaDeFirma
* idPrestamoPreconcedido
* cantidadAmortizadaMensualmente

PagosPrestamos:



* numpago
* fechaPago
* cantidadPago
